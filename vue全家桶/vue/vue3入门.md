- 通过vite创建项目

  ```bash
  npm i -g create-vite-app
  create-vite-app vue3-vite-demo
  ```

- 通过webpack创建项目

  ```bash
  npm i -g @vue/cli
  # 对于 Vue 3.x 的项目，需要使用 Vue CLI v4.5 以上的版本
  # 查看vue-cli版本
  vue -V 
  vue create vue3-demo
  ```
