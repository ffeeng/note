1. 用nginx启静态服务

   ```bash
   # linux启动nginx
   systemctl start nginx.service 
   # mac 启动nginx
   sudo nginx
   ```

   需要安装nginx ，参考：[nginx](http://nginx.org/)

2. python启静态服务

   ```bash
   #Python 2.x 
   python -m SimpleHTTPServer  
   #Python 3.x 
   python -m http.server
   ```


​      注：需要安装python，mac或linux环境可能自带python

3. node启静态服务
   ```bash
   npm install http-server -g
   http-server . -p 8000
   ```
